\begin{center}
    \Large{\textbf{ABSTRACT}}\\
\end{center}
\vspace{1cm}

Coverage path planning is an important task in autonomous robot systems, especially when a robot must traverse an area for safety-critical operations. In this thesis, the application context is a ground robot operating in hazardous or difficult-to-access areas. The robot needs to cover the working area while considering limited energy, non-uniform terrain, static obstacles, dynamic obstacles, and the ability to return to its base for recharging or safe mission termination.

This thesis develops a grid-based coverage robot simulation system with energy constraints and dynamic obstacles. The environment is represented as a weighted grid map, where each traversable cell may have a different movement cost. The path planner is based on oriented Dijkstra search, in which each state includes both the robot position and heading. Before selecting a new target cell, the robot checks whether it can still return to the base, reducing the risk of moving deep into the working area without enough remaining energy to return.

The final result is a C++ and OpenCV simulation program that visualizes robot behavior, records logs, exports benchmark data, and supports comparison between the model with turning cost and an ideal omnidirectional baseline. The final benchmark includes scenario groups for base position, energy capacity, obstacle density, obstacle structure, dynamic obstacles, weighted terrain, map size, and the omnidirectional assumption. Experimental results show that turning cost, return-to-base energy, map structure, and dynamic obstacles directly affect coverage behavior and mission outcome.
